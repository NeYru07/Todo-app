# Laravel ToDo App (Docker/Sail)

Простое приложение для управления задачами, созданное на Laravel 11, PostgreSQL и Tailwind CSS.

## Функционал
- Добавление, редактирование и удаление задач.
- Пометка задачи как выполненной (автоматическая фиксация времени).
- Адаптивная верстка на Tailwind.

## Как запустить (через Docker Sail)
1. Установите зависимости (если проверяете локально):
   `composer install`
2. Запустите контейнеры:
   `./vendor/bin/sail up -d`
3. Выполните миграции базы данных:
   `./vendor/bin/sail artisan migrate`
4. Приложение будет доступно по адресу: `http://localhost`
